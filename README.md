## 🌠 Overview

**Cosmic Note** is a visual knowledge graph and spatial second brain. It breaks free from traditional folders and lists, allowing you to organize your thoughts exactly how they connect in your mind. 

Notes are stars in an infinite galaxy canvas; relationships between notes are constellation lines (edges). Built entirely offline-first for extreme speed, privacy, and fluidity.

### ✨ Key Features

- 🌌 **Infinite Galaxy Canvas:** Fluid panning and zooming powered by `React Flow` with a stunning, dynamically generated nebula background.
- 🔗 **Constellation Mapping:** Drag and drop connections between notes to map relationships visually.
- 🎨 **Premium Aesthetics:** 4 meticulously curated color themes (Deep Space, Purple Nebula, Aurora, Solar Eclipse) with glassmorphism panels and dynamic `framer-motion` micro-animations.
- ⚡ **Offline-First & Blazing Fast:** Zero latency. No accounts, no cloud sync required. Everything persists securely in your local browser storage.
- ⌨️ **Keyboard Centric:** Fully featured hotkeys (Ctrl+N, Ctrl+F, Delete, Space+Drag) designed for power users.
- 📦 **Data Portability:** Complete import and export pipeline for your workspace. Your data is yours forever.

## 🛠️ Technology Stack

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Canvas Rendering**: [@xyflow/react](https://reactflow.dev/) (React Flow)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

You will need **Node.js** (v18 or higher) and **npm** installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/yourusername/cosmic-note.git
   cd cosmic-note
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to the local host address provided in the terminal (usually `http://localhost:5173/`).

---

## 📦 Deployment

Building Cosmic Note for production is completely straightforward since it's a 100% client-side application.

1. **Build the production bundle:**
   ```bash
   npm run build
   ```

2. **Preview the production build locally (optional):**
   ```bash
   npm run preview
   ```

3. **Deploy:**
   The output in the `dist` folder can be hosted statically on any modern platform such as:
   - [Vercel](https://vercel.com/)
   - [Netlify](https://netlify.com/)
   - [GitHub Pages](https://pages.github.com/)
   - [Cloudflare Pages](https://pages.cloudflare.com/)
